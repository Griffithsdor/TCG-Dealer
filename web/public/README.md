# public/

Vite sirve estos archivos en la raíz (`/`).

## Logo de marca

Guarda la imagen del logo que enviaste aquí, con este nombre exacto:

```
web/public/tccheese-logo.png
```

Se usa en el header (`src/Logo.tsx`) y como favicon (`index.html`).
Mientras el archivo no exista, el header cae automáticamente al símbolo SVG de respaldo,
así que la app nunca se rompe. Al colocar el PNG, aparece solo — sin más cambios.
